Изменение температуры за несколько дней.

Одной из множества функций программы является представление статистической информации о температурах за несколько дней в виде визульного объекта- графика. График "Изменение температуры за несколько дней" показывает динамику температуры за 10 ближайших дней, для любого выбранного города. По оси X расположены дни с их календарным значением. По оси Y расположены значения средних температур за каждый день текущего периода.

Чтобы увидеть этот график следует под окошком температур нужного города нажать на кнопку "Несколько дней". Новое окно с графиком мгновенно откроется
