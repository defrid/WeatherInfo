Описание установки

Открыть файл с расширением .exe, поэтапно выполнить процесс установки .

Для запуска программы необходимо иметь на компьютере .NET Framework версии 4.0 и выше. Корректная работа программы обеспечивается при подключении к интернету, а также в следующих операционных системах:

  • *Windows 7
  • *Windows Vista
