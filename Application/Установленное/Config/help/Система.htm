Система.

Данный блок отвечает за выборя языка приложения и за его автоматический запуск. По умолчанию программа имеет русскоязычный интерфейс и работает без автозапуска с Windows.

Чтобы изменить язык приложения нужно:

в выпадающем списке рядом с подписью "Язык" выбрать нужный либо английский, либо русский язык. Затем нажать на кнопку "Ок". Программа сохранит изменения и автоматически обновится.

Чтобы программа автоматически запускалась при включения компьютера следует:

в данном блоке в настройках поставить галочку напротив поля "Автозапуск с Windows". Затем нажать на кнопку "Ок". Программа сохранит изменения и автоматически обновится.
