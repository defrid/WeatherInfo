Изменение температуры за день.

Одной из множества функций программы является представление статистической информации о температурах за день в виде визульного объекта- графика. График "Изменение температуры за день" показывает динамику температуры за текущий день, для любого выбранного города. По оси X расположены часы суток от 0 до 24. По оси Y расположены значения температур за каждый час текущего дня.

Чтобы увидеть этот график следует под окошком температур нужного города нажать на кнопку "Один день". Новое окно с графиком мгновенно откроется
